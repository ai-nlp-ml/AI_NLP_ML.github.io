Shared Tasks



  • ST1: TechDOfication 2020 : Technical DOmain Identification (https://ssmt.iiit.ac.in/techdofication.html)


For this task, participants are asked to develop system/s that automatically identify the technical domain of a given text (a small passage) in specified Langauge (English, Bangla, Gujarati, Hindi, Malayalam, Marathi, Tamil, Telugu). Such a piece of text provides information about specific Coarse grained technical domains (subTask-1*) like Computer Science, Physics, Life Science, Law etc or the Fine grained subdomains (subTask-2*) for Computer Science domain, it can be of Operating System, Computer Network, Database etc.


Paper Submission:

Submission is electronic, using the SoftConf START conference management system. The submission site is now available at https://www.softconf.com/icon2020/techdofication_papers/


More Details: https://ssmt.iiit.ac.in/techdofication.html



  • ST2: Unsupervised Technical Domain Terms Extraction (TermTraction - 2020) (https://ssmt.iiit.ac.in/TermTraction.html)


Automatic extraction of domain-specific terms based on relevant domain concepts for a given document is a highly challenging task. Domain Term extraction can be defined as the automated process of identifying terminology from domain specific texts. Domain terms are described as textual span which represent/describe the domain. Participants are asked to develop an unsupervised system/s which automatically identify the technical terms from the given text (a document) of English. Such a document provides information about specific technical domains like Computer Science, Physics, Life Science, Law etc.


Paper Submission:

Submission is electronic, using the SoftConf START conference management system. The submission site is now available at https://www.softconf.com/icon2020/termtraction_papers/


More Details: https://ssmt.iiit.ac.in/TermTraction.html



  • ST3: Adap-MT 2020: Low Resource Domain Adaptation for Indic Machine Translation (https://ssmt.iiit.ac.in/machinetranslation.html)


The focus of this task is to show capability of general domain machine translation when translating into Indic languages (English - Hindi , English - Telugu and Hindi - Telugu) and Low resource domain adaptation of MT systems using (Existing available parallel corpora + small indomain parallel data) for AI and Chemistry Domains.


Paper Submission:

Submission is electronic, using the SoftConf START conference management system. The submission site is now available at https://www.softconf.com/icon2020/AdapMT_papers/


More Details: https://ssmt.iiit.ac.in/machinetranslation.html
